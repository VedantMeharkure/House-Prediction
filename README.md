# 🏠 House Price Prediction (Ridge vs Lasso)

## 🚀 Overview

This project predicts house prices using Machine Learning models:

* Ridge Regression
* Lasso Regression

It includes an interactive Streamlit web app with visualizations and real-time predictions.

---

## 📊 Features

* Interactive UI with sliders
* Model comparison (Ridge vs Lasso)
* Feature importance visualization
* Prediction history tracking
* Dark/Light theme

---

## 🧠 Dataset

Boston Housing Dataset
Features used:

* RM (Rooms)
* LSTAT (% lower income)
* PTRATIO (student-teacher ratio)
* INDUS (industrial area)

---

## 📈 Results

* Ridge R²: ~0.73
* Lasso R²: ~0.72

---

```bash
pip install -r requirements.txt
streamlit run app/app.py
```
